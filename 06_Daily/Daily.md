# 📅 Daily

> **`06_Daily/` 폴더의 입구 노트입니다.** 매일의 작업 기록이 여기에 모입니다.

← [[🏠 Home]]

## ⚡ 오늘 노트 빠르게 열기

- 단축키: `Ctrl+P` → "오늘의 일일 노트" (또는 "Open today's daily note")
- 캘린더 플러그인 설치 시: 우측 사이드바 캘린더에서 날짜 클릭

## 📅 최근 7일 (자동)

```dataview
LIST file.mtime
FROM "06_Daily"
WHERE date >= date(today) - dur(7 days)
SORT date DESC
```

## 🔗 데일리 노트 활용법

### 매일 아침
1. 데일리 노트 열기
2. **오늘 할 일**에 3~5개만 (욕심 금지)
3. **하루 동안 회의/작업할 때마다** 본문에 `[[프로젝트명]]` 링크

### 퇴근 전
1. 미완료 작업은 내일로 옮기거나 → 프로젝트 `tasks.md`로
2. **3줄 회고**: 잘한 것 / 아쉬운 것 / 내일 첫 일

### 주말
1. 한 주 데일리 노트 훑기
2. [[자기개발]] 회고로 정리
3. `00_Inbox` 비우기
