## 💡 Resources vs Areas vs Projects

| 구분 | 예시 |
|------|------|
| Project | "이번 달에 React 책 한 권 끝내기" |
| Area | "프론트엔드 역량 꾸준히 관리" |
| **Resource** | "React Hook 정리 노트" (참고용 자료) |

Resource는 **읽고 참고하는 자료** 자체. 행동이 아니라 정보 저장소.
