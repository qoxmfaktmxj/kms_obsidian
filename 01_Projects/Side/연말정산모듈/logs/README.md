# 📂 연말정산모듈 - 작업 로그

> 이 폴더는 **날짜별 작업 기록**을 모으는 곳입니다.

## 사용법

- 작업한 날마다 노트 생성: `2026-04-20.md` 형식
- `_Templates/WorkLog` 템플릿 사용
- 상위 [[연말정산모듈]] MOC에서 자동으로 모임

## 데일리 노트와의 차이

| 데일리 노트 | 작업 로그 |
|-------------|-----------|
| 그날 **전체** 기록 | 이 프로젝트 **만** 기록 |
| `06_Daily/`에 위치 | 각 프로젝트 폴더의 `logs/` |
| 짧고 가벼움 | 기술적 디테일 OK |

> 둘 다 쓸 필요는 없음. 가벼운 사이드는 데일리만, 회사 큰 프로젝트는 작업 로그도.
