# 📂 HR 하네스 업그레이드 - 작업 로그

> 이 폴더는 **날짜별 작업 기록**을 모으는 곳입니다.

## 사용법

- 작업한 날마다 노트 생성: `2026-04-20.md` 형식
- `_Templates/WorkLog` 템플릿 사용
- 상위 [[HR하네스-업그레이드]] MOC에서 자동으로 모임

## 작성 가이드

기술 프로젝트 작업 로그엔 다음 항목을 강력 추천:

```markdown
## 한 일
- [구현/수정한 코드, 회의 결과 등]

## 결정/발견
- [왜 이 선택을 했는지 - 미래의 나에게 보내는 메모]

## 막힌 점
- [해결 못 한 것, 추후 확인 필요한 것]

## 다음
- [ ] [내일 첫 일]
```

**"결정/발견"이 핵심**입니다. 코드는 git에 남지만, **왜 그렇게 했는지는 여기에만** 남습니다.
