# LLM Wiki Index

이 공간은 raw 자료를 그대로 저장하는 곳이 아니라, repo/회의/문서/블로그 분석 결과를 읽고 편집된 지식으로 축적하는 compiled wiki다.

## 원칙

1. raw source는 진실원천이 아니다.
2. wiki page는 현재 이해의 정본이다.
3. 불확실한 내용은 confidence를 낮춘다.
4. 모순은 숨기지 않고 contradictions.md에 기록한다.
5. stale page는 stale-pages.md에 기록한다.
